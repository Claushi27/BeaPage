## Features

## How to Use
1. Open `index.html` in a web browser
2. Answer the interactive questions
3. Read our love story
4. Try to click "No" (spoiler: you can't!)
5. Click "Yes" to start the relationship counter

## Live Day Counter
The day counter starts when you click "YES" and persists forever in your browser's localStorage.
